# Loan Calculator Project

Java console application that calculates loan payments and analyzes how extra payments impact total interest paid and loan payoff time.

## Features
- Calculates monthly loan payments
- Compares repayment scenarios with and without extra payments
- Measures interest savings and reduced payoff time

## How to Run

### Clone the Repository
```bash
git clone https://github.com/kennybac360/LoanCalculatorProject.git
```

### Compile the Program
```bash
cd groupproject2025
javac LoanTemplate.java TestLoanClassTemplate.java
```

### Run the Program
```bash
cd ..
java groupproject2025.TestLoanClassTemplate
```
